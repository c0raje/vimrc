.vimrc

Mi .vimrc

Configuracion basica y de forma nativa:


set number " habilita los numeros de lineas

set nonumber " deshabilita los numeros de lineas

set  relativenumber " habilitan los numeros relativos de las lineas

set showmode " muestra el modo en que se encuentra vim

set ruler " muestra la posicion del cursor

set ignorecase " busca las mismas letras en una palabra, ya sean mayusculas o minusculas, ahora (palabra) sera lo mismo que (Palabra)
